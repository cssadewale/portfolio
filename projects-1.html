CBT Pro (CBT.ng)

"Built by a tutor, for tutors. Not with web dev skills — with clear thinking and AI leverage."

  • ProblemNigerian teachers had no free, reliable digital exam tool. Paper CBT was slow, error-prone, and inaccessible to ordinary teachers without technical backgrounds.
  • SolutionFull-stack CBT platform: 7-column CSV question upload, timed sessions with auto-submit, per-student question randomisation, anti-cheat tab-switch detection, flag-for-review, auto-marking, result certificates with WhatsApp share, 4-tab teacher analytics dashboard (Overview, Question Analysis, Student Comparison, Trends), optional Gemini AI explanations per answer.
  • ImpactLive with real secondary school students today. Built entirely on an Android tablet using Acode editor — not because I learned web development, but because I think clearly as a data scientist and leveraged AI to operate competently in an unfamiliar technical field. Zero budget. Zero compromise. This is what AI-Augmented Solutions Development looks like in practice.

Platform Features
  • CSV question upload — no manual entry. 7-column format with optional AI explanations
  • Timed exam sessions with countdown timer and auto-submit
  • Per-student question and option randomisation (anti-copy)
  • Anti-cheat tab-switch detection with configurable tolerance
  • Flag-for-review during exam — revisit questions before submitting
  • Auto-marking + result certificate with WhatsApp share
  • 4-tab analytics: Overview, Question Analysis, Student Comparison, Trends
  • Optional Gemini API AI explanations per question
