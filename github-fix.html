🔑 How to Create a GitHub Personal Access Token
1
Go to github.com/settings/tokens — make sure you are logged in as cssadewale
2
Click "Generate new token" → choose "Generate new token (classic)" (not fine-grained — classic is simpler)
3
In the Note field, type: Portfolio Admin
4
Set Expiration to No expiration (or 1 year). Then under Select scopes, tick only repo — the top checkbox. That gives access to your repositories.
5
Scroll down and click "Generate token". Copy the token — it starts with ghp_. You can only see it once. Paste it above immediately.
⚠ Common mistakes: Using a fine-grained token (use classic instead) · Not checking the repo scope · Token expired · Typing username/repo with wrong capitalisation · Using the Cloudflare domain (cssadewale.pages.dev) instead of the actual GitHub repo name
