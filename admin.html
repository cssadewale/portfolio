How to Use This Admin Panel
Everything you need to know to manage your portfolio without writing code.
The Workflow (3 Steps)

Step 1 — Edit — Use the panels in the sidebar to edit your projects, story, skills, certifications, journey milestones, and contact info. Everything has a form — no code needed.

Step 2 — Export — When you are done editing, go to Export & Deploy and click Download portfolio-data.json. This file contains all your content.

Step 3 — Deploy — Upload the downloaded portfolio-data.json file to your GitHub repo (cssadewale/portfolio). Cloudflare Pages will auto-deploy in under 60 seconds.

Changing Your PIN

Open admin.html in a text editor. Find the line that says const ADMIN_PIN = 'hmg2026' near the bottom of the file. Change hmg2026 to any password you want. Save and re-upload to GitHub.

Frequently Asked Questions

Will my changes appear immediately?
Yes — once you upload the new portfolio-data.json to GitHub and Cloudflare Pages redeploys (30–60 seconds), your changes are live.

What if I close the browser?
Your edits are saved in your browser's local storage automatically as you type. They will still be there when you come back — on the same device and browser.

Can I add a completely new section?
Yes — go to any panel, add content, then use Export. The portfolio pages automatically read from portfolio-data.json on load.

How do I hide a section without deleting it?
Go to Show / Hide Sections in the sidebar and toggle the section off. It will be hidden on the live site but your content is preserved.

GitHub Upload Steps (on Android)

1. Go to github.com/cssadewale/portfolio in your browser

2. Tap Add file → Upload files

3. Select the portfolio-data.json file you just downloaded

4. Scroll down and tap Commit changes

5. Wait 30–60 seconds — Cloudflare Pages auto-deploys

6. Visit your live site to confirm changes ✅
